# Task5 Search Task
## protected internal
## private protected

## 1- كلمة protected تستخدم علشان لو عاوز اخلي الشئ دا يقتصر رؤيته في نفس الكلاس او الكلاس الوراثه منه 
## 2- كلمة internal تستخدم علشان لو عاوز الشئ يقتصر رؤيته في نفس المشروع(Assembly)
## 3- كلمة protected internal تستخدم علشان اقصر رؤية الشئ دااخل الكلاس الوارث في نفس المشروع او في مشروع اخر  
## 4- كلمة private protected تستخدم علشان اقصر رؤية الشئ دااخل الكلاس الوارث في نفس المشروع فقط
